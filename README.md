# Dicas de Skin Care Coreano

Bem-vindo ao guia completo de skin care coreano! Este site oferece dicas práticas sobre a rotina K-beauty, desde limpeza de pele até os melhores produtos, com cores suaves em rosa e lilás para uma experiência relaxante.

## Sobre o Projeto
- **Tema**: Beleza e cuidados com a pele inspirados na Coreia do Sul.
- **Tecnologias**: HTML, CSS e JavaScript (site estático, sem backend).
- **Funcionalidades**: Slider de produtos, quiz interativo para tipo de pele, timer para máscaras, modo escuro e rolagem suave.

## Como Usar
1. Acesse o site hospedado em: [Sua URL aqui, ex: https://seu-usuario.github.io/skin-care-coreano/](https://seu-usuario.github.io/skin-care-coreano/).
2. Navegue pelas seções: Introdução, Quiz, Rotina, Produtos, etc.
3. Use o quiz para descobrir seu tipo de pele e personalize a rotina.

## Instalação Local (para desenvolvimento)
1. Clone o repositório: `git clone https://github.com/seu-usuario/skin-care-coreano.git`
2. Abra o arquivo `index.html` em um navegador (não precisa de servidor, pois é estático).
